# Vanilla-PvP-16x
Vanilla PvP 16x: A high-performance resource pack that preserves the classic look, refined for competitive Minecraft Bedrock gameplay.
## Features
- **Classic Aesthetic:** Keeps the original game style.
- **Optimized for PvP:** Enhanced visual clarity.
- **16x Resolution:** Improved performance.

## Installation
1. Download the latest `.mcpack` file from the [Releases](https://github.com/spzxn/Vanilla-PvP-16x/releases) page.
2. Open the file to import it into Minecraft.
3. Enable the pack in your world or global settings.
